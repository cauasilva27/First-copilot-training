🕹 Modos do Copiloto (Study, Ask, Plan e Agent)

Através dos diferentes modos de copiloto, você poderá aumentar sua produtividade e aprendizado, basta escolher qual ferramenta mais se adequa à sua necessidade na hora de elaborar seu projeto.

📖 Study

Focado em ensinar, o modo Study é ideal para seu aprendizado ativo.

Nele, você não terá só as respostas, mas sim:

o raciocínio por trás delas
exemplos funcionais e analogias de fácil entendimento
perguntas reflexivas para fixação de conhecimento
abordagem que parte do nível mais básico, progredindo de acordo com o dev

Prompt:

❓ Ask

O modo Ask é pensado para responder suas dúvidas e lhe ajudar a compreender o que está acontecendo, sem alterar ou implementar algo em seu código ou projeto. 

Nele, você poderá:

perguntar algo sobre seu projeto ou código
abordar uma questão de uma parte específica, como um arquivo ou erro

Prompt:

📝 Plan

O modo Plan é ideal para lhe dar um direcionamento ao longo do seu projeto. Nele, será estabelecido um:

objetivo claro
passo a passo
ferramentas/tecnologias que melhor lhe atendem
resultado esperado

Prompt:

🤖 Agent

O modo Agent age como um executor, não só explicando, mas agindo e tomando iniciativa junto de você. Ao definir um objetivo, ele irá operar dentro do seu projeto:

criando arquivos
alterando estruturas
e entregando códigos prontos para serem implementados
